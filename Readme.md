### 사용법


1. node / npm 설치
2. 필요 패키지 설치
~~~
npm install -g
~~~
3. api 폴더에 있는 .env.sample을 .env로 변경(또는 .env 파일 생성)
4. .env 파일에 API 키 등 환경변수 입력
5. 백엔드 서버(api/server.js) 및 프론트엔드 서버 실행(public/index.html)
